# Unity_ZombieFPS
3D FPS zombie gun shooting game<br>

Genre: FPS<br>

<h4>Function</h4>
- Move, Fire and jump with animations<br>
- Weapon change system<br>
- Item get and use system<br>
- Zombie enemy attack and die<br><br>
